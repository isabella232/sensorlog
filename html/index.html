SensorLog

Мобильный сборщик данных окружающей среды с помощью Android и Arduino

Arduino и датчики

Позволяет получать данные окружающей среды со следующих сенсоров:

  1. MQ2 C4H10 (изобутан)
  2. MQ4 CH4 (концентрация метана)
  3. MQ7 CO (угарный газ)
  4. Датчик шума

Android приложение

Позволяет получать данные с Arduino (и не только) и сохранить их в лог.
